### VITE

Vite es un bundler y dev server moderno pensado para frontend (React, Vue, Svelte, etc.), que te ayuda a compilar, recargar y optimizar tu código de cliente. Solo lo necesitas si estás trabajando en el frontend moderno y quieres esas ventajas.

### EXPRESS

Express es un framework de Node.js para crear servidores y manejar rutas, APIs y middlewares. Puedes usarlo solo, sin necesidad de ningún bundler o herramienta extra.
