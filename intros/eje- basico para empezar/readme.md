### → Consejos para Principiantes

• Practica diariamente - Aunque sea 30 minutos
• Construye proyectos - Teoría + práctica
• Lee código de otros - GitHub es tu amigo
• No te frustres con errores - Son parte del aprendizaje
• Usa console.log() - Para debuggear tu código
