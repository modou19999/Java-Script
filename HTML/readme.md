# GRID
display → Activa el modo de diseño del elemento (ej: grid convierte el elemento en contenedor grid).

grid-template-columns → Define el número y tamaño de las columnas.

grid-template-rows → Define el número y tamaño de las filas.

gap → Define el espacio entre filas y columnas.

justify-content → Alinea toda la cuadrícula horizontalmente dentro del contenedor.

align-items → Alinea verticalmente los elementos dentro de sus celdas.

grid-column → Indica en qué columna empieza y termina un elemento.

grid-row → Indica en qué fila empieza y termina un elemento.

# ◣ Para alinear horizontal
    display: grid;
    grid-template-columns: repeat(3, 1fr);
    gap: 5px; 
# ◥

# FLEX
display → Activa el modo Flexbox (display: flex convierte el elemento en contenedor flexible).

flex-direction → Define la dirección de los elementos (fila o columna).

justify-content → Alinea los elementos horizontalmente (eje principal).

align-items → Alinea los elementos verticalmente (eje secundario).

flex-wrap → Permite que los elementos bajen a otra línea si no caben.

gap → Define el espacio entre los elementos.

flex → Define cuánto crece, se reduce o ocupa un elemento.

flex-grow → Indica cuánto puede crecer un elemento.

flex-shrink → Indica cuánto puede reducirse un elemento.

flex-basis → Define el tamaño base inicial del elemento.

align-self → Permite alinear un elemento individual diferente a los demás.
